Project Goal

The Turing Machine (TM) is the foundation of modern computation. While most simulators focus on simple string acceptance (like palindromes), they fail to demonstrate how complex logic stems from simple transitions.

Our goal was to bridge this gap by creating an interactive emulator that visually demonstrates TM-based arithmetic (Addition, Subtraction, Multiplication) step-by-step.

Key Features

  • Dual Mode: Supports both Unary and Binary systems.
  • Visual Tape: Real-time head tracking and tape expansion.
  • Transition Logs: Explains why a move occurred in plain English.
  • Python Core: Logic implemented using Python generators for efficiency.
